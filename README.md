# Инфраструктура защищенного удаленного доступа для UI/UX компании

## 📝 Описание проекта
В рамках данного кейса реализуется построение защищенной корпоративной сети для компании, занимающейся UI/UX-дизайном. Основная цель — обеспечить безопасную работу удаленных сотрудников через VPN, исключив риски при использовании публичных Wi-Fi сетей.

Данный репозиторий содержит инструменты для **автоматизированного развертывания** первого уровня защиты — собственного Удостоверяющего Центра (CA).

## 🛠 Технологический стек и инфраструктура
* **ОС:** Ubuntu 20.04 LTS (или аналоги на базе Debian).
* **Безопасность:** Firewalld (настройка портов 22/tcp для управления и 1194/tcp для VPN).
* **PKI:** Easy-RSA (автоматизированное управление ключами и сертификатами).

## 📦 Описание файлов
* `setup_pki.sh` — Bash-скрипт для автоматической настройки Firewall и инициализации Root CA в режиме `batch` (без запроса паролей).
* `setup-server.deb` — Пользовательский deb-пакет, включающий в себя скрипт настройки и описывающий необходимые зависимости (`easy-rsa`, `firewalld`).

## 🚀 Инструкция по развертыванию
1. **Обновите списки пакетов:**
   ```bash
   sudo apt update
   ```
2. **Установите подготовленный пакет:**
   ```bash
   sudo apt install ./setup-server.deb
   ```
3. **Запустите автоматическую настройку:**
   ```bash
   sudo setup_pki.sh
   ```

## 💎 Артефакты блока
После выполнения команд в системе будет создан корневой сертификат:
* `/root/easy-rsa/pki/ca.crt` — Корневой сертификат удостоверяющего центра.
